UNIVERSITAS CONTOH

Beranda Pendidikan Penelitian Pengabdian Fakultas Kontak

1. Inovasi Kecerdasan Buatan dalam Dunia Pendidikan

Dr. Andi Prasetyo, M.T., Rina Maharani, S.Kom

Artikel ini membahas bagaimana penerapan kecerdasan buatan (AI) telah mengubah cara belajar dan mengajar di institusi pendidikan. Penggunaan chatbot, sistem rekomendasi, dan evaluasi otomatis dibahas secara mendalam.

2. Pengembangan Material Nano untuk Energi Terbarukan

Dewi Kartika, Ph.D., Budi Santoso, M.Sc

Fokus penelitian ini adalah pengembangan material nano untuk meningkatkan efisiensi panel surya dan penyimpanan energi. Studi eksperimental menunjukkan peningkatan efisiensi hingga 20%.

3. Sistem Monitoring Kualitas Udara Berbasis IoT

Ilham Nurrohman, S.T., M.T.

Penelitian ini mengembangkan prototipe sistem monitoring kualitas udara menggunakan sensor IoT yang terintegrasi dengan aplikasi seluler untuk pelaporan secara real-time.

4. Pengaruh Algoritma Machine Learning terhadap Prediksi Cuaca Mikro

Siti Zahra, M.Sc

Penelitian ini mengkaji efektivitas berbagai algoritma machine learning dalam memprediksi cuaca mikro di wilayah perkotaan menggunakan dataset dari BMKG dan sensor lokal.

5. Revolusi Teknologi Blockchain dalam Dunia Kesehatan

Rizky Aditya, S.Kom

Studi ini mengeksplorasi penggunaan teknologi blockchain untuk mengamankan data medis, meningkatkan transparansi, dan mengurangi biaya administrasi di rumah sakit.
